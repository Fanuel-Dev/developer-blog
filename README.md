# developer-blog
A full developer blog platform, styled like a git commit log — that's the design hook: posts read like commits (hash, diff stats, branch-style tags) instead of generic blog cards. 

What's functional:

Feed with live search and tag filtering (click any tag pill)
Post detail view with formatted body, code blocks, like button, and comments (add your own)
New post modal — write a title, tags, and content, and it publishes instantly to the top of the feed with auto-computed read time and diff stats
Sidebar with tag cloud, trending posts by likes, and maintainer card
Fully responsive, keyboard-friendly (Esc closes the modal)
It's a single self-contained HTML file — open it directly in any browser, no build step. Everything runs in memory, so a refresh resets the seeded data (five real sample posts) but you're free to wire it up to a real backend later if you want persistence.
